# Быстрый старт: Система обучения

## Что было создано

Полная система обучения для игры с:
- ✅ Всплывающими подсказками
- ✅ Подсветкой кнопок и элементов UI (с эффектом мигания)
- ✅ Пошаговыми инструкциями
- ✅ Сохранением статуса обучения
- ✅ Автоматической инициализацией

## Быстрая интеграция (3 шага)

### Шаг 1: Добавьте TutorialSetup на Canvas
1. Откройте сцену в Unity Editor
2. Выберите Canvas
3. Нажмите "Add Component" → TutorialSetup

### Шаг 2: Настройте параметры
- Включите `Enable On Gameplay` для игровой сцены
- Включите `Enable On Main Menu` для главного меню

### Шаг 3: Готово!
Обучение будет автоматически запущено при первом запуске игры.

## Как это работает

1. **TutorialSetup** запускается на старте сцены
2. **TutorialManager** создаёт все необходимые компоненты (если их нет)
3. **TutorialInitializer** подготавливает сценарий для текущей сцены
4. Подсказки показываются по одной, с подсветкой целевых элементов

## Кастомизация

### Изменить текст подсказок
Откройте `TutorialInitializer.cs` и отредактируйте методы:
- `InitializeMainMenuTutorial()` - для главного меню
- `InitializeGameplayTutorial()` - для игры

### Добавить новый шаг
```csharp
_tutorialInitializer.AddStep(
    "Текст подсказки",
    targetButton,
    waitForClick: false,
    duration: 2f
);
```

### Стилизация
Измените в `TutorialSetup.CreateTutorialSystem()`:
- Цвета и размеры шрифта
- Цвет подсветки в `TutorialHighlight.cs`
- Размер и позицию панели подсказок

## Отладка

**Обучение не запускается?**
- Убедитесь, что `TutorialSetup` добавлен на Canvas
- Проверьте имя сцены (должно быть "MainMenu" или "CatSpawn")
- Откройте консоль (Ctrl+Shift+C в Unity) на предмет ошибок

**Сбросить обучение:**
```csharp
// В консоли (Ctrl+K для открытия командной строки):
TutorialManager.ResetTutorial();
```

Или добавьте кнопку в главное меню:
```csharp
if (GUILayout.Button("Переиграть обучение"))
{
    TutorialManager.ResetTutorial();
}
```
