## 🚀 Как использовать

### Вариант 1: Автоматически (🎯 РЕКОМЕНДУЕТСЯ)
1. Откройте **Canvas** в сцене
2. Добавьте компонент **TutorialSetup**
3. Включите флаги для нужных сцен
4. **✨ Готово!** Обучение работает

### Вариант 2: Вручную
```csharp
// Создайте менеджер
var tutorialManager = gameObject.AddComponent<TutorialManager>();

// Добавьте шаги
tutorialManager.AddStep("Текст подсказки", targetButton, duration: 2f);

// Запустите
tutorialManager.StartTutorial();
```

## 🔧 Как кастомизировать

### Изменить тексты
Отредактируйте `TutorialInitializer.cs`:
```csharp
InitializeMainMenuTutorial()
InitializeGameplayTutorial()
```

### Изменить дизайн
В `TutorialSetup.CreateTutorialSystem()`:
- Цвет фона панели
- Размер и стиль шрифта
- Цвет подсветки
- Позиция панели подсказок

### Добавить новый шаг
```csharp
tutorialManager.AddStep(
    "Текст подсказки",
    targetButton,
    waitForClick: false,
    duration: 2f
);
```

## 🐛 Отладка

**Обучение не запускается?**
- ✓ Проверьте `TutorialSetup` на Canvas
- ✓ Убедитесь в имени сцены (MainMenu / CatSpawn)
- ✓ Откройте консоль на предмет ошибок

**Сбросить обучение:**
```csharp
TutorialManager.ResetTutorial();
```
